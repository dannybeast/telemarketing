Главная/Услуги/Привем SMS

Виртуальные номера для приема SMS

Услуга, которая позволяет принимать SMS-сообщения на закрепленный за вами номер. Используйте услугу для двустороннего обмена информацией между вами и вашими клиентами, например, для проведения викторин или опроса по качеству работы

Преимущества

  • Бесплатное подключение для ряда номеров
  • Онлайн-статистика в личном кабинете через интернет-браузер
  • Отправка уникального ответного SMS-сообщения
  • Возможность выделения «Золотых номеров»
  • Высокие проценты выплат
  • Последующая выгрузка данных в CSV
  • Передача данных на внешний скрипт по HTTPS или SMPP-протоколам для дальнейшей обработки
  • Отправка данных на заданный E-mail
  • Индивидуальный "красивый" короткий номер
